
# SAR Tournament Manager

A simple browser-based tournament calculator for Super Animal Royale.

## Features

* Paste `/getplayers` output directly
* Automatic scoring (kills + placement)
* Multiple rounds
* Team grouping system
* Solos and Teams mode
* Final leaderboard with placements
* Custom scoring settings
* Background color & gradient customization
* 
## Notes

* Works entirely in browser (no install needed)
* Data is saved locally

## Made for Super Animal Royale tournaments


## HOW TO WORK: EASIEST WAY!

Step 1: Download the file as ZIP

Step 2: Open the code in notepad

Step 3: Press File.

Step 4: Press Save As

Step 5: Make sure the file is named as: "sar_tournament_manager.html" MAKE SURE THERE IS NO .txt BEHIND IT!

Step 6: Make sure the Save As Type is All files!

Step 7: Go to files on your desktop, and search for "sar_tournament" The file should pop up

Step 8: Right click the file, and press open with Google, Microsoft, etc. ONLY APPS LIKE GOOGLE OR MICROSOFT OR THE APPLICATION WONT WORK!

Step 9: Follow the steps below this

## HOW TO USE? 

Step 1: Press "Add Tournament" on the website, and name your tournament

Step 2: Make your scoring! 

Step 3: After a round of a tournament you host, type /getplayers to get your code! Then, paste it in the text box where it says "Paste /getplayers output here..."

Step 4: Press "Add Round" you will get your scoring

Step 5: Continue doing rounds as needed.

Step 6: Once finished, press the "Finished!" button. This will save your tournament!

Step 7: (OPTIONAL) Press the "GROUP PEOPLE" button AFTER you paste the /get players and create the round. You can press players, and the column with their nameand stats will turn blue. Press the "GROUP PEOPLE" button after doing all of that, then put the team name
